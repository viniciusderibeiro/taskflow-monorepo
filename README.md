# TaskFlow

This project is a Kanban task management system built as a monorepo for learning purposes. It is designed to cover both fullstack development and agile methodologies in a practical way.

## Stack

- **Backend:** Java 21 + Spring Boot 3, PostgreSQL, Redis, JWT  
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, React Query  
- **Infrastructure:** Docker Compose
